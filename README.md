# GTM+GA4 자동화 서비스 기획 로그
> 작성일: 2026-05-30 | 카이파이(Caify) 신규 서비스 기획

---

## 1. 서비스 개요

### 아이디어
채팅 + AI 에이전트 방식으로 고객이 원하는 GTM/GA4 설정을 자연어로 말하면,
구글 공식 문서를 탐색하고 GTM에 자동으로 태그/트리거/변수를 생성 및 적용해주는 서비스.

### 핵심 가치
- 마케터/소상공인이 GTM+GA4를 말만 해도 세팅됨
- 코드 한 줄 몰라도 데이터 추적 가능
- 세팅 후 지속적인 성과 리포트 제공

---

## 2. 기술 구조

### 전체 파이프라인
```
[채팅 입력] → [AI 에이전트] → [GTM 설정 JSON 생성] → [GTM API 자동 적용]
```

### GTM API v2
- 공식 Google API 존재 (Tag Manager API v2)
- 태그/트리거/변수 생성, publish까지 API로 완전 자동화 가능
- DOM 자동화(셀레니움) 불필요

### 인증 방식
- OAuth 2.0 (고객이 "Google 로그인"으로 동의)
- GCP에서 앱 1번 등록 → client_id + client_secret 발급
- 개발/테스트: 심사 없이 즉시 가능
- 상용화: OAuth 앱 심사 (1~2주)

### 부업 모델에서의 인증 구조
```
고객이 GTM에 caifyhelp@gmail.com 편집자 추가
→ 우리 계정 1개로 모든 고객 GTM 접근 가능
→ API 키 고객마다 발급 불필요
```

## 3. 서비스 로직 전체 플로우

### 사전 준비 (1회)
```
GCP OAuth 앱 등록 → client_id + client_secret → Google 계정 로그인
```

### 의뢰 접수 시 고객에게 받는 것
1. GTM에 편집자 추가 요청
2. 사전 질문지: 어떤 이벤트, 플랫폼(카페24/고도몰/워드프레스), 발생 시점, 추적 데이터

### 툴 실행 플로우
```
1단계: 고객 GTM 컨테이너 선택
2단계: 채팅으로 요구사항 입력
3단계: AI 처리 (의도 파악 → 문서 탐색 → JSON 구성)
4단계: 결과물 2가지 생성
  A. GTM 설정 자동 적용 (변수→트리거→태그→publish)
  B. 데이터레이어 스니펫 + 삽입 가이드
5단계: 완료 리포트 전달
```

### 서비스 범위
| 항목 | 자동화 | 수동 |
|------|--------|------|
| GTM 태그/트리거/변수 생성 | ✅ API | |
| GTM publish | ✅ API | |
| 데이터레이어 코드 생성 | ✅ AI | |
| 삽입 가이드 | ✅ 자동 | |
| 실제 코드 삽입 | | ✅ 고객 직접 |

## 4. 데이터레이어

### 개념
웹사이트와 GTM 사이의 데이터 전달 창구.
GTM이 알 수 없는 비즈니스 데이터(결제금액, 상품명 등)를 사이트 코드에서 밀어주는 역할.

### 케이스 분류
- **데이터레이어 있는 고객**: GTM API로 바로 세팅 가능
- **데이터레이어 없는 고객**: 단순 이벤트만 가능, 스니펫 생성 후 고객이 직접 삽입

### 코드 삽입 방법 (플랫폼별)
| 플랫폼 | 삽입 위치 |
|--------|-----------|
| 카페24 | 관리자 → 스크립트 관리 |
| 고도몰 | 관리자 → 레이아웃 관리 |
| 워드프레스 | WPCode 플러그인 |
| Shopify | theme.liquid |
| 직접 개발 | 개발자 전달 |

## 5. GA4 Admin API 연동

GA4도 API 존재. GTM과 함께 자동화 가능.

| GA4 설정 | API 가능 여부 |
|----------|-------------|
| 전환 이벤트 마킹 | ✅ |
| 커스텀 이벤트 등록 | ✅ |
| 사용자 속성 정의 | ✅ |
| 잠재고객 생성 | ✅ |
| GA4 속성 신규 생성 | ✅ |

## 8. 카이파이(Caify)와의 연계

### 카이파이 현재 서비스
- 블로그 자동화: 30만원/월
- 홈페이지 제작: 30만원/월
- 숏폼 영상 자동화: 30만원/월
- 타겟: 마케팅 담당자 없는 소상공인

### 연계 구조
```
GTM+GA4 세팅 (저관여, 1회성)
    ↓ 신뢰 형성
GA4 데이터로 "블로그/영상 유입 부족" 확인
    ↓
카이파이 블로그/숏폼 제안
    ↓
카이파이 구독 → 더 풍부한 성과 데이터
    ↓
루프: 콘텐츠 생산 → 성과 측정 → 콘텐츠 개선
```

### GTM 서비스의 카이파이 내 역할
1. **영업 채널**: 저가 서비스로 고객 확보 → 카이파이 업셀
2. **이탈 방지**: 성과 가시화로 구독 해지 명분 제거
3. **포지셔닝 강화**: "마케팅 실행 + 성과 측정" 풀사이클

## 9. 리포트 구조 (A안 기준)

### 커뮤니케이션 구조
| 타이밍 | 내용 | 채널 |
|--------|------|------|
| 즉시 (GTM Webhook) | 전화/카카오 클릭 알림 | 카카오 알림톡 |
| 매주 월요일 | 지난주 3줄 요약 | 카카오 알림톡 |
| 매월 1일 | 종합 분석 + 액션 | 카카오 or 이메일 |
| 이상 감지 시 | 트래픽 급변 알림 | 즉시 알림 |

### 월간 리포트 포함 내용
1. 이번 달 3줄 요약
2. 채널별 유입 분석 (블로그/SNS/검색/직접)
3. 콘텐츠별 성과 (UTM 기준)
4. 관심 신호 (전화클릭/카카오클릭/폼제출)
5. 방문자 행동 (체류시간/이탈률/모바일비율)
6. 이상 감지
7. 다음달 액션 3가지

### 리포트 샘플
```
📊 5월 성과 요약

✅ 이번 달 347명이 홈페이지에 왔어요 (지난달+23%)
   그 중 12명이 전화번호를 눌렀어요
   블로그가 가장 큰 유입 채널이에요 (68%)

📍 어디서 왔나
   네이버 블로그  68% (235명)
   인스타그램     21% (73명)
   네이버 검색    8%  (28명)

🏆 콘텐츠 1위: "강남 냉삼 맛집" → 89명 유입

⚠️ 3주차 방문자 -40% → 블로그 발행 없었던 주

📌 다음달 액션
   1. "강남 회식 장소" 주제 글 작성
   2. 모바일 전화버튼 상단 이동 (83%가 모바일)
   3. 인스타 화요일 오전 발행 (유입 최다)
```

## 10. 기술적 한계 및 해결책

### 한계
| 항목 | 한계 | 해결 |
|------|------|------|
| 전화 문의 | 클릭만 추적, 실제 통화 불가 | A안: 클릭으로 대체, B안: 콜트래킹 |
| GA4 딜레이 | 표준 보고서 24~48시간 | GTM Webhook으로 우회 |
| 카카오 상담 | 클릭만 추적 | 클릭으로 대체 |

### 실시간 알림 구조 (GTM Webhook)
```
GTM 이벤트 발생
→ GA4 전송 + 동시에 우리 서버 Webhook
→ 우리 서버 → 카카오 알림톡 (수초 내)
```

## 11. 방향성 A vs B

### A안 (범위 축소)
- 세팅 + 클릭 기반 리포트 + 주간/월간 요약
- 빠른 출시 가능 (1~2개월)
- 가격: 세팅 3.9만, 리포트 2.9만/월

### B안 (완전 추적)
- A안 + 콜 트래킹 + 카카오/네이버 API 연동
- 실제 문의/통화 완료까지 추적
- 구축 기간 3~4개월
- 가격: 세팅 9.9만, 리포트 7.9만/월

### 추천 전략
```
A안으로 빠르게 출시 → 고객 50명 확보
→ 데이터 쌓이면 B안으로 진화
→ 업종별 벤치마크 구축 (진짜 차별점)
```
